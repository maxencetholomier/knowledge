# Zettelkasten Method

Zettelkasten, meaning "note castle" in German, is a note-taking method.

The principles are:

- Atomic note-taking (one idea per note)
- No hierarchy between notes (no subdirectories)
- Links between different notes

It helps resolve certain [problems](problem_with_classic_note_taking.md) of classical note-taking approaches.

It is not better than other methods, it also has [disadvantages](problem_with_zettlekasten.md), it's more adapted to certain situations.

For more information: [Wikipedia - Zettelkasten](https://en.wikipedia.org/wiki/Zettelkasten)
